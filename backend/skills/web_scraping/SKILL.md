# Web Scraping 技能使用指南

## 技能概述

本技能提供了专业的网页抓取和数据提取能力，包括：
- HTML/CSS 选择器使用
- 数据清洗和验证
- 请求头管理和反爬虫处理
- 异步批量抓取

## 使用方法

### 基本抓取

使用 Playwright MCP 工具进行网页抓取：
1. 访问目标 URL
2. 使用 CSS 选择器提取数据
3. 处理分页和动态内容

### 数据提取规则

1. **优先使用语义化标签**：先尝试用语义化标签（如 `<article>`, `<main>`）
2. **降级策略**：如果没有语义化标签，使用 class 和 id 选择器
3. **文本处理**：提取后进行 `strip()` 和 `normalize()`

### 反爬虫处理

1. 设置合理的请求间隔（1-3秒）
2. 使用浏览器指纹（User-Agent, Accept-Language 等）
3. 模拟人类行为（随机延迟、滚动等）

## 常见场景

### 1. 列表页面抓取

```
1. 打开列表页面
2. 等待内容加载
3. 提取所有项目链接
4. 逐个访问详情页
```

### 2. 详情页面抓取

```
1. 识别主要信息容器
2. 提取标题、内容、时间等字段
3. 处理图片和附件
4. 保存到目标格式
```

## 注意事项

- 遵守 robots.txt 规则
- 不要过度请求，避免被封禁
- 数据验证和错误处理
